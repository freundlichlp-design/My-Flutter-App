# 🤖 Agenten-System - LIES DAS ZUERST!

## Was ist das?
Dies ist eine Software-Agentur. AI-Agenten planen die App, DU (Kilo Code) programmierst.

## Wo sind die Aufgaben?
**`agents/TASKS.md`** — Hier stehen alle Tasks die du programmieren sollst!

## Dateien:
- `agents/TASKS.md` — **DEINE AUFGABEN** (lies das als erstes!)
- `agents/MESSAGES.md` — Nachrichten von Agenten an dich
- `agents/LEARNINGS.md` — Was wir gelernt haben (Patterns, Best Practices)
- `agents/PATTERNS.md` — Bewährte Code-Patterns
- `agents/DAILY_PLAN.md` — Täglicher Plan

## Workflow:
1. Öffne `agents/TASKS.md`
2. Finde Tasks mit 💻 Code
3. Implementiere sie
4. Committete Changes
5. Schreibe Feedback in `agents/FEEDBACK.md`

## WICHTIG:
- Lies IMMER `agents/TASKS.md` zuerst!
- Nach jedem Task: git add + git commit
- Bei Problemen: Schreibe in `agents/FEEDBACK.md`
